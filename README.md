# SheSure: AI-Based PCOS Detection System

SheSure is a multimodal AI system designed to assist in early detection of Polycystic Ovary Syndrome (PCOS). The project combines two machine learning approaches:

1. **MLP (Multi-Layer Perceptron)** for tabular medical data
2. **CNN (Convolutional Neural Network)** for ultrasound image analysis

The goal is to build a screening system that analyzes both clinical data and ultrasound images to predict PCOS risk.

This project is part of the Design Thinking and Innovation course.

---

## Project Architecture

The system follows a multimodal approach:

User Input  
↓  
Tabular Data → MLP Model  
Ultrasound Image → CNN Model  
↓  
Prediction Layer  
↓  
PCOS Risk Output
